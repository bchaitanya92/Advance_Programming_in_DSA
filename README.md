## üöÄ Getting Started

### 1. Prerequisites
- C compiler (e.g., GCC, MinGW, or any standard C compiler)

### 2. Compile & Run
Open PowerShell or your terminal and run:
```powershell
gcc Program_1_avl_tree.c -o avl_tree.exe
./avl_tree.exe
```
Replace `Program_1_avl_tree.c` with any file you want to run.
